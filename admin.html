📌 使用说明

1. Base URL：填写上游服务的根地址，例如 https://api.example.com

2. API Key：填写分配给你的 X-API-Key，用于上游服务鉴权

3. 配置保存到 data/settings.json，重启服务后同样有效

4. 仪表盘会显示当前配置是否已就绪
